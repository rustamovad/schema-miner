We view MatrixNet as a backbone that is an improvement over FPN. We demonstrated the impact of using Ma-trixNet for one-stage object detection, which can be extended in the future to two-stage object detection. Ma-trixNets can also replace FPNs in other computer vision tasks such as instance segmentation, key-point detection, and panoptic segmentation tasks. Code for the paper is available at https://github.com/arashwan/ matrixnet.
